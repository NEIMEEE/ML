#  Spam Mail Prediction using Machine Learning

## Description
Ce projet a pour objectif de construire un modèle de **Machine Learning** capable de prédire si un email est **spam** ou **non-spam**.  
Il s’appuie sur des techniques de **traitement du langage naturel (NLP)** et des algorithmes de classification pour analyser le contenu textuel des emails.

##  Technologies utilisées
- **Python 3.x**
- **Jupyter Notebook / Google Colab**
- Bibliothèques principales :
  - `pandas` : manipulation des données
  - `numpy` : calculs numériques
  - `scikit-learn` : modèles de Machine Learning
  - `matplotlib` & `seaborn` : visualisation des résultats
